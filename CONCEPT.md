## アプリ名

**「だが、それでいい」**

---

## コアコンセプト

### メインコンセプト
「毎日の習慣化を否定する」のではなく、**「できなくても、別のことができていればいいじゃん」** をユーザーに伝えるアプリ。

- 習慣化できているなら、それは最高
- 習慣化できなかった日も、別のことができていれば、それでいい
- 「できなかった」を責めるのではなく、「別のことができた」を称える

### 裏コンセプト
人をダメにしているように見えるが、本来の目的とは別の「他のこと」をやっているため、**結果的にダメにはなっていない状態**を作る。

### UXのイメージ
> 英語の勉強ができた日も、できなかった代わりに筋トレした日も、散歩しただけの日も、全部「今日も動いた」。
> ユーザーは英語が喋れないかもしれないが、代わりにマッチョになる。それでいい。

---

## 従来アプリとの違い

| 従来の習慣トラッカー | だが、それでいい |
|---|---|
| 一つの習慣をストイックに継続させる | どんな行動も無駄じゃない |
| できなかった日 = 失敗 | できなかった日も別のことができた = 成功 |
| ストリーク（連続記録）を重視 | 何かしら動いたことを称える |
| 目標に縛られる | 目標はあくまで出発点 |

## ユーザーペルソナ & システムへのインプット

### プロフィール登録（AIサポートあり）
- 氏名・年齢・職業・興味分野・生活リズム（朝型/夜型）・現在の悩み
- 登録後、AIがプロフィールを解析して初期Pivot_Goal候補を自動生成

### 目標（Goal）設定
- 複数設定可能（最大10件）
- 例：「英語を習慣的にやりたい」
- **Goalはあくまで出発点。そこに沿う必要はない。別の強みが開花するかもしれない。**
- Primary_Goal（最優先）を1件指定。残りは自動的にPivot_Goal候補になる

### 状態検知トリガー（v1はアプリ起動時のみ）
- 位置情報：ずっと家にいる（外出していない）
- スクリーンタイム/学習ログ：今日一日、Goal関連アプリを使っていない
- 心理状態：「ダラダラしてたら何もしないで時間が過ぎた」などをアプリに入力

## コアワークフロー

### Step 1: アプリ起動時のレコメンド
- アプリを開いたタイミングで状態を検知し、提案を表示
- **初回は必ずPrimary_Goalに関連した提案**を出す
- 提案は常に「未来の自分（Future_Self_Model）」の語り口で行う
- 例：「カフェに移動して英語の文法勉強しない？」
- ボタン：「やる」 / 「今日は別のことをやる」

> ⚠️ v1ではアプリを開いていないタイミングでのプッシュ通知はしない（v2以降）

### Step 2: Yes/No分岐とピボット
**「やる」を選んだ場合（Yes）:**
- Primary_Goalの達成をサポートする詳細なアクションステップを表示
- そのまま目標をこなす → 英語が喋れるようになる

**「今日は別のことをやる」を選んだ場合（No）:**
- 即座にPivot機能を起動
- Pivot_Goalに基づく別の行動提案を「未来の自分」の語り口で提示
- ボタン：「これをやる」 / 「別の提案を見る」
- 最大3回までピボット可能
- 最終的に「5分だけ外の空気を吸いに行く」などの最低限の行動を提案

**学習機能:**
- この時の行動結果を基に、次回からのレコメンド内容を最適化

### Step 3: プロフィールの動的更新
- 行動の過程・結果からプロフィールを随時更新
- 連続3日以上Pivot_Goalをこなすと、そのGoalをPrimary_Goal候補として昇格提案
- 結果（Noルートの場合）：英語は喋れないが、代わりにマッチョになる

### Step 4: 努力ポイント報酬
- Primary_Goal完了：10ポイント
- Pivot_Goal完了：7ポイント
- 最低限の行動（外出・散歩など）：3ポイント
- 何もしなかった日：ポイントなし + 「明日また何かできるよ」という前向きメッセージ

## 特徴的な機能

### Future Self Model（未来の自分の再現）
**仕組み:**
- 同じようなステータス・プロフィールを持つ他ユーザーの行動・結果データ（匿名化）を収集
- そのデータを基に「未来の自分」モデルを構築
- 「あなたと似た状況だった人は、筋トレを始めて3ヶ月でこうなった」という実データ裏付きのアドバイスを実現

**ポイント:**
- 類似ユーザーが5件未満の場合はProfileとGoalのみの推定モデルで代替
- Action_Logが蓄積されるほど精度が上がる

### Persona_Voice（未来の自分が語りかける）
- すべての提案・メッセージは「未来の自分」が語りかけるスタイル
- 一人称「俺（私）」、ユーザーへの呼びかけ「お前（あなた）」の口語体
- トーン選択：「厳しめ」「普通」「優しめ」
- Primary_Goal達成日：「今日もできたね、最高」
- Pivot_Goal達成日：「英語できなかったけど、筋トレできたじゃん。それでいいよ」

### ピボット機能
- アプリを開いたタイミングで実行（v1）
- 一つの習慣に固執せず、即座に別の行動を提案
- 筋トレ → 健康的になる
- カフェに行く → 散歩することになる
- などの連鎖的な価値を提示

## 用語集

| 用語 | 説明 |
|---|---|
| `Goal` | ユーザーが設定する目標。あくまで出発点 |
| `Primary_Goal` | 最優先のGoal。Recommendationの出発点 |
| `Pivot_Goal` | Primary_Goalが難しい日の、同じくらい価値ある別の選択肢 |
| `Flexible_Achievement` | どんな行動も価値があるという本アプリの中心的価値観 |
| `State_Detection` | アプリ起動時にユーザーの状態を検知する機能 |
| `Trigger` | State_Detectionが検知した、提案を発火させる条件 |
| `Recommendation` | AIが生成する行動提案。初回は必ずPrimary_Goal関連 |
| `Pivot` | Recommendationを拒否した際にPivot_Goalへ切り替える機能 |
| `Effort_Point` | 何かしら行動した日に付与されるポイント |
| `Persona_Voice` | 「未来の自分」が語りかけるスタイルの通知・提案トーン |
| `Future_Self_Model` | Similar_User_Dataを基に構築した「未来の自分」モデル |
| `Similar_User_Data` | 類似プロフィールの他ユーザーの匿名化された行動・結果データ |
| `Learning_Engine` | 行動結果を蓄積しRecommendationを最適化するAIコンポーネント |
| `Action_Log` | ユーザーが実際に行った行動の記録 |
